# ModelGate

ModelGate is a curated index of official subscription entry links for popular large language models. It helps developers quickly find and access the right pay-to-use AI service — whether for chat, coding, multimodal tasks, or API integration — without digging through confusing portals.

> Currently covers **8 major Chinese LLM providers** (May 2026).

### Which guide should I read?

- **If you are a non-Chinese citizen** → start with the [English guide](guide/China-AI-Model-Portal-Guide.md). It highlights which platforms require a Chinese ID card or mainland phone number for registration, and points you toward alternatives that accept international sign-up.
- **If you are a Chinese citizen** → read the [Chinese guide](guide/中国主流大模型入口导航.md) for accurate real-name verification (实名认证) rules and domestic account registration instructions.
