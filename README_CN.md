## 痛点

你在搜索引擎里输入**"通义千问 API"**。

前三行结果给了三个不同域名：`qianwen.com`、`qwen.com`、`bailian.aliyun.com`——域名、页面、登录方式完全不同，却看起来都跟通义千问有关。

点第一个，是聊天页。有对话框，有推荐问题，但翻遍整个页面找不到 API Key 在哪里。没有开发者入口。没有控制台。

再点第二个。模型介绍页。技术博客。跑分图表。还是找不到 API Key。

第三个终于对了——百炼开发者控制台，真正的 API 入口。但它的域名挂在阿里云下面，登录是另一套流程，界面让人觉得这是两个完全不同的公司。

十五分钟过去了。你算幸运的。你找到了。

**通义千问不是唯一一家。** 百度把聊天、API、AI Studio 拆成三套独立账号，互不相通。字节跳动的豆包聊天和火山方舟 API 分属两个企业体系，账号完全隔离。腾讯混元的聊天用微信/QQ 登录，API 却要另开腾讯云账号。MiniMax 更是有四套——聊天、国内 API、国际 API、星野角色扮演，每套独立注册。

每个想用国产大模型做开发的工程师，都在这里撞过墙。每一个。

---

## ModelGate 做什么

ModelGate 是一份手动维护的**直链索引**——每家主流国产大模型厂商的聊天入口、API Key 申请页、API 文档、Agent 平台、编程入口、多模态入口，一个页面全部列清。没有博客软文，没有营销落地页，没有错误转向。

每个厂商的**账号体系**也一并梳理清楚，因为找到对的链接只解决了一半问题。
